# Question

Implement the generic version of `Array.push`

For example:

```ts
type Result = Push<[1, 2], '3'>; // [1, 2, '3']
```

# Answer

```ts
/* _____________ Your Code Here _____________ */

type Push<T extends unknown[], U> = [...T, U];
```
